# Echo Throne

*Kill the god. Claim the throne. Fight the echo.*

Boss Rush em pixel art top-down: enfrente Deuses Corrompidos presos nas Arenas, extraia Sangue Primordial para fortalecer seu corpo — e pague o preço da corrupção. Na run seguinte, o último desafio pode ser **você**, o eco de quem venceu antes.
